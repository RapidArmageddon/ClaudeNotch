# ClaudeNotch v1.1 — Chimes + Project Names

## Feature 1: Audio Chimes

Play a macOS system sound when Claude needs your attention.

**Triggers:**
- `waitingForInput` (query complete) → `Tink`
- `error` → `Basso`
- Tool permission needed → `Pop`

No sound on `.launching`, `.processing`, `.idle`.

Toggle via menu bar: "Sounds" on/off. Default: on. Persisted via `UserDefaults`.

## Feature 2: Project Name Display

Show which project Claude is working in, extracted from the log line:
`Starting local session local_xxx in /path/to/project`

**Display rules:**
- Single session: show project name during `.launching`, then normal state labels
- Multiple active sessions: alternate project name and state label every 2 seconds

Monitor tracks session IDs and their associated project names.

## Feature 3: Menu Bar Settings

Menu items:
- ClaudeNotch (title)
- Separator
- Sounds (toggle, checkmark)
- Show Project Name (toggle, checkmark)
- Separator
- Quit

Both persisted via `UserDefaults`.
